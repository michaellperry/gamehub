1. Navigate to FusionAuth Admin

Click the link below to open the FusionAuth administration interface:

2. Create Admin Account

Follow the setup wizard to create your administrator account. This will be your main admin login for GameHub.

3. Generate API Key

In the FusionAuth admin panel, run the setup to create an API key. Skip the other setup steps for now.

4. Copy API Key and Run Setup Script

Then run this command in your terminal:

./scripts/setup-fusionauth.sh <API_KEY>
1. Navigate to Tenant Management

Click the link below to open the tenant management interface:

â†’ Open Tenant Management

2. Log in with FusionAuth Credentials

Use the admin account you created in Step 1 to log into the tenant management portal.

3. Create New Tenant

Create a new tenant and copy the generated public key that appears after creation.

4. Configure Tenant with Setup Script

Then run this command in your terminal:

./scripts/setup-tenant.sh "<TENANT_PUBLIC_KEY>"
1. Navigate to Service Principal Management

Click the link below to open the service principal interface:

â†’ Open Service Principal Management

2. Log in with FusionAuth Credentials

Use the same admin account from previous steps to log into the service principal management.

3. Create Service Principal for Player-IP

Click "Create Service Principal for Player-IP". The system will automatically:

  • Fetch the public key from http://localhost/player-ip/public-key
  • Create the service principal with the fetched key
  • Update service configurations automatically
4. Confirm Creation

Verify that the service principal was created successfully. No additional terminal commands are needed for this step.
